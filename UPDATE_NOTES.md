# Pokémon 카드깡 시뮬레이터 — 업데이트 노트

최신 메이저 배포 기준 (2026-05 시점)의 누적 기능·밸런스·버그픽스 정리.

---

## 🎮 핵심 플레이 컨텐츠

### 박스 · 팩 개봉
- **6개 세트** — m2a 메가드림ex · m2 인페르노X · sv8 초전브레이커 · sv2a 151 · sv8a 테라스탈 페스티벌 · sv5a 크림슨 헤이즈
- **여러 박스 한번에** (×3 / ×5 / ×10) — 자동 개봉 후 집계 결과만 표시
- **팩 확률 재설계** (세트별 Hit 슬롯) — `AR 10 : SAR 5 : UR 1 : MUR 0.5` 비율 준수
- **localStorage 박스 지속성** — 구매 후 뒤로가기/새로고침해도 팩 그리드 복원
- **저장 안정성** — `persistPackWithRetry` 3회 재시도 (0.5s → 1s → 1.5s 백오프), 실패 시 자동 환불 (`refund_box_purchase`), "카드 저장 중…" 명시적 로딩

### PCL 감별 (구 PSA)
- **AURA → SSS → PCL** 브랜드 정착
- 감별 대상: **SR · MA · SAR · MUR · UR** (5종)
- 성공률 30% (6·7 → 10까지 분포), 실패 시 카드 손상 + 디스코드 알림
- **일괄 감별** 지원 (Mac 작업분) — 여러 장을 한 번에 감별
- **감별 실패 애니메이션** — 카드가 흔들리고 균열이 가며 조각으로 분해되는 shatter 연출 (1.8초)
- **슬랩 UI** — 실제 PSA 케이스 스타일: [PCL 브랜드 | 카드정보 | 큰 등급숫자] 단일 행 헤더 + 등급 라벨(GEM MINT 등) + 카드창 + 하단 cert + 바코드
- **아크릴 케이스 테두리** — 3px 투명 아크릴 프레임으로 물리적 슬랩 질감

### 내 포켓몬센터 🏛️
- 센터 배경 이미지 (포켓몬 센터 내부 탑뷰)
- **6×6 그리드** (36슬롯) — 보관함 자유 배치
- **4종 보관함** — 기본 2만 · 유리 7만 · 프리미엄 25만 · 레전더리 70만p (모두 1슬롯)
- **전시 조건**: PCL 9·10 슬랩만 전시 가능
- **시간당 수익**: `희귀도 베이스 + PCL 보너스`
  - SR 1k · MA 1.5k · SAR 3k · UR 5k · MUR 7k  
  - PCL 9 +2k · PCL 10 +5k
  - 조합 예: MUR 10 = 12,000p/hr · SR 9 = 3,000p/hr
- **정보 모달** — 4탭(보관함 · 전시·수익 · 부수기 · 랭킹)으로 규칙 전부 정리

### 부수기 (Sabotage)
- **초대 링크**로 친구 센터 방문
- 보관함별 공격 비용: 3만 / 5만 / 10만 / 20만p
- 기본 성공률 30% − 보관함 방어 = 실효
- **방어 %** — basic 5% · glass 12% · premium 22% · legendary 28% (레전더리 실효 2%)
- 성공 시 보관함·슬랩 영원히 소멸 + **보관함가의 80% 전리품** + 랭킹 +100점
- 실패 시 비용 환불 없음
- **디스코드 웹훅** 자동 알림 + 센터 주인 페이지에 **📜 방문 기록** 로그
- 피해자 랭킹에서 파괴된 PCL 점수 자동 차감

### 야생 배틀 🌿
- **/wild** 라우트 — PCL 슬랩 보유 시에만 진입
- 야생 포켓몬 15마리 (이상해씨 · 피카츄 · 뮤츠 등) — PokeAPI BW 애니메이션 GIF
- **18타입 상성 매트릭스** 완비 (×2 / ×1 / ×0.5 / ×0)
- HP · ATK = 희귀도 × PCL 등급 배율 (MUR 10 = HP 190 / ATK 48)
- 턴제 전투 · 말풍선 캐릭터 양옆 배치 · 데미지 플로팅 · 상성 메시지
- **랜덤 전장** 7종 — 풀숲/동굴/해변/화산/설산/밤의 숲/고대 유적 (순수 CSS 그라디언트)
- 승리 시 `야생 HP × 100p` 보상 (50,000p 상한)
- **패배 = 슬랩 영구 삭제** — 작별 대사 · 파스스 사라지는 연출 · 30초 쿨다운

## 💰 경제 · 랭킹

### 지갑 일괄판매 (상인보다 약 1/40)
- C 25 · U 50 · R 100 · RR 200 · AR 500 · SR/MA 1k · SAR 3k · UR 5k · MUR 10k
- PCL 6·7 1k · 8 2k · 9 10k · 10 20k
- **PCL 슬랩 일괄판매** — 판정 등급별 즉시 판매, 전시 중 슬랩은 자동 제외
- **5,000장 상한** (PCL 1,000개 별도) — `record_pack_pull` / `submit_psa_grading` 서버에서 cap 검사, 초과 시 환불 플로우 자동 동작 + 사용자에게 정확한 한글 메시지 노출

### 상인
- MUR 100만 · UR/SAR 60만 · SR 40만 · MA/AR 20만 · RR 10만 · 그 외 5만p
- 시간당 판매 5회 · 교체 5회 제한
- **NPC 몰입 개편** — 냥트레이더 상인 이미지 + 상태별 반응 대사 + 타이핑 말풍선

### 선물
- **하루 5회** 제한
- 수신자 Navbar에 **빨간 배지** (미확인 pending 수)
- /gifts 방문 또는 수락/거절 시 배지 자동 해제 (`viewed_at` 컬럼)

### 랭킹 (사용자 랭킹)
- **두 탭** — 🏆 랭킹 점수 · ⚔️ 전투력
- **랭킹 점수** 공식:
  - PCL 6·7 +100 · 8 +150 · 9 +350 · 10 +500
  - 부수기 성공 +100 (등급 무관)
  - 파괴 시 자동 차감 (psa_gradings 삭제로 자연스럽게)
- **전투력** 공식:
  - `rarity_power × pcl_power` 합산
  - rarity_power: SR 5 · MA 6 · SAR 7 · UR 8 · MUR 10
  - pcl_power: 9 → 9 · 10 → 10
  - MUR 10 한 장 = 100 전투력
- **🔥 조롱하기** 버튼 (전투력 탭 전용) — 수신자에게 실시간 팝업 (15초 polling)
- 행마다 🏛️ 센터 버튼 (상대 센터 방문)

## ✨ UX · 시각 개편

### 공통
- **PageHeader** 컴포넌트로 모든 페이지 타이틀 · 부제 · stats 통일
- **NpcDialog** — 무드별 애니메이션 (idle bob / working wiggle / excited bounce / sad sway / shocked shake) + 타이핑 말풍선
- **Portal** — 모든 모달을 `document.body`로 탈출 → z-index/stacking 충돌 해결
- 모달 z-index `z-[60]` → `z-[100]` 로 표준화
- MUR/SAR 전용 카드 이펙트:
  - **MUR**: 대각선 금박 shimmer (3.4s) + 금빛 inset glow
  - **SAR**: 회전 홀로그래픽 conic + 별자리 twinkle
  - 나머지 등급은 깔끔하게 렌더

### 감정실
- 후딘 박사 NPC + 상태별 반응 대사 (감별 중 / 성공 / 실패 각기 다름)
- 10등급 시 "놀라워요! 완벽한 GEM MINT 판정!" 식 특별 대사

### 카드 상세
- **🔗 카드 공유하기** — `navigator.share({ files: [file] })`로 카드 이미지 첨부
- `/api/card-image/[id]` 프록시 라우트 (CORS + 캐시 + OG 이미지 겸용)
- `/card/[id]` OpenGraph 메타 — 카카오톡/디스코드에 카드 art 미리보기

### 관리자 (hun 전용)
- 👑 Nav 칩 (모바일 포함)
- `/admin` — 유저별 포인트 지급/차감 + 전체 유저 +10만 / +100만 일괄 지급 버튼
- 서버 RPC가 `user_id = 'hun'` 검증 → 다른 계정 시도는 403

## 🐛 주요 버그 픽스 (역사순)

- 팩/박스 열고 확인 안 해도 카드 유실 X (박스 구매 시점에 전체 풀 DB 기록)
- 감별 실패 애니메이션 후 카드 잔상 남던 문제 (phase 조건 순서 수정)
- PsaSlab 고정 폭 → 반응형 w-full max-w, 모달 좁은 컬럼에서 짤림 해결
- 센터 보관함 shadow가 body `overflow-x-hidden`에 걸려 잘리던 문제 수정
- 모달 z-index 충돌 해결 (Portal 도입)
- PCL만 있어도 일괄 판매 진입 가능 (psa.length 포함해서 hasAny 판정)
- 멀티박스 중간 실패 시에도 앞서 성공한 카드는 결과 화면에 표시
- 조롱 받은 사람 실시간 팝업 (15초 polling + 탭 복귀 시 즉시 fetch)
- hun 모바일에서도 👑 어드민 버튼 노출
- 지갑 PCL 탭에 전시 중 슬랩도 배지로 표시 (숨기지 않음)
